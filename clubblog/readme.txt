== Description ==
 
A Free WordPress Blog / Magazine / Personal / Writers Theme. ClubBlog is a clean minimal and responsive WordPress theme well suited for fashion, writers, travel, health, business, finance, portfolio, design, art, photography, personal or any other creative websites and blogs. Developed with bootstrap framework that makes it mobile and tablets friendly. Get free support at https://ask.thewpclub.com/

Theme Link: https://www.thewpclub.com/clubblog-wordpress-theme/
Demo Link : https://demo.thewpclub.com/clubblog/

== Installation ==
 
1. Upload the zip file to your themes directory.
2. Apply the theme.
3. Go to Appearance > Customize in main menu and configure your theme's settings.
